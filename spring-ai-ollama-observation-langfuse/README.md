## spring-ai-ollama-observation-langfuse

> 演示 Spring AI 与 **Langfuse** 可观测性集成（Ollama 场景）。版本由父 POM 管理。

## 官方文档对应

- [Observability](https://docs.spring.io/spring-ai/reference/observability/index.html)
- [Spring AI Reference — Introduction](https://docs.spring.io/spring-ai/reference/index.html)

### 运行

```bash
cd spring-ai-ollama-observation-langfuse
mvn spring-boot:run
```

按 `application.yml` 配置 Langfuse 地址与密钥。
