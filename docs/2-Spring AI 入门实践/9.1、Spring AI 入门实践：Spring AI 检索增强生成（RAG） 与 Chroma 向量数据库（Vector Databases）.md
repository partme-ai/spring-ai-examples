# Spring AI 入门实践：Spring AI 与 Chroma 向量数据库（Vector Databases）

> 基于 Spring AI 框架实现与 Chroma 向量数据库的集成，提供文档向量存储、相似度搜索、过滤查询等功能，为 RAG 应用提供向量检索支持。

---

## 一、项目概述

### 1.1 项目定位

本项目是 Spring AI 框架下集成 Chroma 向量数据库的示例，展示了如何在 Java/Spring Boot 应用中实现文档的向量存储、相似度检索和高级查询功能。

### 1.2 技术栈

| 组件 | 版本 | 说明 |
|------|------|------|
| Spring Boot | 3.5.6 | 基础框架 |
| Spring AI | 1.1.4 | AI 能力集成 |
| Chroma | - | 向量数据库 |
| spring-ai-starter-vector-store-chroma | - | Spring AI Chroma 集成 |

### 1.3 代码地址
**GitHub**：https://github.com/partme-ai/spring-ai-examples/tree/main/spring-ai-ollama-rag-chroma
**本地路径**：`spring-ai-ollama-rag-chroma/`

### 1.4 核心功能

- ✅ 文档向量存储：支持将文档转换为向量并存储
- ✅ 相似度搜索：基于向量相似度检索相关文档
- ✅ 过滤查询：支持元数据过滤条件
- ✅ 批量操作：支持批量添加、删除文档
- ✅ 自定义嵌入：支持配置不同的嵌入模型

## 二、Chroma 向量数据库简介

### 2.1 Chroma 介绍

Chroma 是一个开源的向量数据库，专为 AI 应用设计，提供简单易用的 API 来存储和检索向量嵌入。Chroma 具有轻量级、易用性强、支持本地部署等特点，非常适合用于 RAG（检索增强生成）应用的开发。

### 2.2 核心特性

| 特性 | 说明 |
|------|------|
| **开源免费** | Apache 2.0 许可证，完全开源 |
| **轻量级** | Python 实现，易于安装和部署 |
| **简单 API** | 提供简洁易用的 Python 和 HTTP API |
| **持久化存储** | 支持本地持久化和内存模式 |
| **集合管理** | 支持多个集合（Collections）的管理 |
| **元数据过滤** | 支持基于元数据的过滤查询 |

### 2.3 部署方式

Chroma 支持多种部署方式：

- **本地模式**：直接在应用中使用，无需独立服务
- **服务模式**：启动独立的 Chroma 服务，通过 HTTP API 访问
- **Docker 部署**：使用 Docker 容器快速部署

## 五、代码实现详解

### 5.1 Chroma 配置类

```java
package com.example.chroma.config;

import org.springframework.ai.embedding.EmbeddingClient;
import org.springframework.ai.vectorstore.ChromaVectorStore;
import org.springframework.ai.vectorstore.VectorStore;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

@Configuration
public class ChromaConfig {

    @Value("${spring.ai.vectorstore.chroma.url:http://localhost:8000}")
    private String chromaUrl;

    @Value("${spring.ai.vectorstore.chroma.collection-name:my-collection}")
    private String collectionName;

    @Bean
    public VectorStore vectorStore(EmbeddingClient embeddingClient) {
        return new ChromaVectorStore(
            embeddingClient,
            chromaUrl,
            collectionName
        );
    }
}
```

### 5.2 向量数据库服务类

```java
package com.example.chroma.service;

import org.springframework.ai.document.Document;
import org.springframework.ai.vectorstore.SearchRequest;
import org.springframework.ai.vectorstore.VectorStore;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;

import java.util.ArrayList;
import java.util.List;
import java.util.Map;

@Service
public class ChromaService {

    @Autowired
    private VectorStore vectorStore;

    public void addDocuments(List<Document> documents) {
        vectorStore.add(documents);
    }

    public List<Document> similaritySearch(String query, int topK) {
        return vectorStore.similaritySearch(query, topK);
    }

    public List<Document> similaritySearchWithFilter(String query, int topK, Map<String, Object> filter) {
        SearchRequest request = SearchRequest.query(query)
            .withTopK(topK)
            .withSimilarityThreshold(0.0)
            .withFilterExpression(filter);
        return vectorStore.similaritySearch(request);
    }

    public void deleteDocuments(List<String> ids) {
        vectorStore.delete(ids);
    }
}
```

### 5.3 REST API 控制器

```java
package com.example.chroma.controller;

import org.springframework.ai.document.Document;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;
import com.example.chroma.service.ChromaService;

import java.util.ArrayList;
import java.util.List;
import java.util.Map;

@RestController
@RequestMapping("/api/chroma")
public class ChromaController {

    @Autowired
    private ChromaService chromaService;

    @PostMapping("/documents")
    public String addDocuments(@RequestBody List<Map<String, Object>> docs) {
        List<Document> documents = new ArrayList<>();
        for (Map<String, Object> doc : docs) {
            String content = (String) doc.get("content");
            @SuppressWarnings("unchecked")
            Map<String, Object> metadata = (Map<String, Object>) doc.get("metadata");
            documents.add(new Document(content, metadata));
        }
        chromaService.addDocuments(documents);
        return "Documents added successfully";
    }

    @GetMapping("/search")
    public List<Document> search(
            @RequestParam String query,
            @RequestParam(defaultValue = "5") int topK) {
        return chromaService.similaritySearch(query, topK);
    }

    @PostMapping("/search/filter")
    public List<Document> searchWithFilter(
            @RequestParam String query,
            @RequestParam(defaultValue = "5") int topK,
            @RequestBody Map<String, Object> filter) {
        return chromaService.similaritySearchWithFilter(query, topK, filter);
    }

    @DeleteMapping("/documents")
    public String deleteDocuments(@RequestBody List<String> ids) {
        chromaService.deleteDocuments(ids);
        return "Documents deleted successfully";
    }
}
```

import org.springframework.web.client.RestTemplate;
import org.springframework.http.HttpEntity;
import org.springframework.http.HttpHeaders;
import org.springframework.http.MediaType;
import java.util.List;
import java.util.Map;

public class ChromaClient {

    private static final String BASE_URL = "http://localhost:8080/api/chroma";
    private RestTemplate restTemplate = new RestTemplate();

    public void addDocuments(List<Map<String, Object>> docs) {
        HttpHeaders headers = new HttpHeaders();
        headers.setContentType(MediaType.APPLICATION_JSON);
        HttpEntity<List<Map<String, Object>>> request = new HttpEntity<>(docs, headers);

        String url = BASE_URL + "/search?query={query}&topK={topK}";
        return restTemplate.getForObject(url, List.class, query, topK);
    }

## 十、常见问题

### Q1: Chroma 服务连接失败？

检查 Chroma 服务是否正常启动，确认配置的 URL 是否正确。可以通过 `curl http://localhost:8000/api/v1/heartbeat` 验证服务状态。

### Q2: 如何更换嵌入模型？

修改 `ChromaConfig` 中的 `EmbeddingClient` Bean 配置，Spring AI 支持多种嵌入模型，如 OpenAI、Azure OpenAI、Ollama 等。

### Q3: 如何实现持久化存储？

Chroma 默认支持持久化存储，数据会保存在服务端的 `./chroma` 目录下。使用 Docker 部署时可以通过 volume 挂载实现数据持久化。

## Java 客户端示例

以下是一个独立的 Java 客户端示例，用于调用 Spring AI Chroma 向量存储服务：

```java
import org.springframework.web.client.RestTemplate;
import org.springframework.http.*;
import java.util.Map;

public class ChromaClient {
    private final RestTemplate restTemplate;
    private final String baseUrl;

    public ChromaClient(String baseUrl) {
        this.baseUrl = baseUrl;
        this.restTemplate = new RestTemplate();
    }

    public Map<String, Object> search(String query) {
        String url = baseUrl + "/chroma/search?query=" +
                    java.net.URLEncoder.encode(query, java.nio.charset.StandardCharsets.UTF_8);
        return restTemplate.getForObject(url, Map.class);
    }

    public Map<String, Object> addDocument(String text, Map<String, Object> metadata) {
        HttpHeaders headers = new HttpHeaders();
        headers.setContentType(MediaType.APPLICATION_JSON);

        Map<String, Object> body = Map.of(
            "text", text,
            "metadata", metadata
        );
        HttpEntity<Map<String, Object>> request = new HttpEntity<>(body, headers);

        ResponseEntity<Map> response = restTemplate.postForEntity(
            baseUrl + "/chroma/add", request, Map.class);

        return response.getBody();
    }

    public static void main(String[] args) {
        ChromaClient client = new ChromaClient("http://localhost:8080");

        // 添加文档
        Map<String, Object> metadata = Map.of("category", "tech", "author", "Claude");
        Map<String, Object> addResult = client.addDocument(
            "Chroma 是一个开源的向量数据库",
            metadata
        );
        System.out.println("Document added: " + addResult.get("id"));

        // 搜索文档
        Map<String, Object> searchResult = client.search("向量数据库");
        System.out.println("\nSearch results: " + searchResult.get("results"));
    }
}
```
