# Spring AI 入门实践：Spring AI 与 Pinecone 向量数据库（Vector Databases）

> 基于 Spring AI 框架实现与 Pinecone 向量数据库的集成，使用托管的 Pinecone 云服务，提供高性能的向量搜索、元数据过滤和 SLA 保障。

本项目是 Spring AI 框架下集成 Pinecone 向量数据库的示例，展示了如何在 Java/Spring Boot 应用中使用托管的 Pinecone 云服务，构建生产级 RAG 应用。

| spring-ai-starter-vector-store-pinecone | - | Spring AI Pinecone 集成 |

**GitHub**：https://github.com/partme-ai/spring-ai-examples/tree/main/spring-ai-ollama-rag-pinecone

## 二、Pinecone 向量数据库简介

### 2.1 Pinecone 介绍

Pinecone 是一个托管的向量数据库服务，专为生产环境设计，提供高性能的向量搜索和管理能力。Pinecone 作为云服务，无需自行维护基础设施，提供企业级的可靠性和 SLA 保障。

### 2.2 核心特性

| 特性 | 说明 |
|------|------|
| **托管服务** | 完全托管，无需管理基础设施 |
| **高性能** | 专为生产环境优化的向量搜索性能 |
| **SLA 保障** | 提供企业级服务水平协议 |
| **元数据过滤** | 强大的元数据过滤和筛选能力 |
| **命名空间** | 支持数据隔离和多租户架构 |
| **自动扩展** | 水平扩展以处理大规模数据 |
| **监控指标** | 提供详细的性能和使用指标 |
| **REST API** | 简单易用的 REST API 接口 |

### 2.3 索引类型

Pinecone 支持多种索引类型，适用于不同场景：

| 索引类型 | 适用场景 | 特点 |
|---------|---------|------|
| **Starter** | 开发测试 | 成本低，适合小规模数据 |
| **Serverless** | 生产环境 | 按使用付费，自动扩缩容 |
| **Pod** | 企业级 | 固定资源，可预测性能 |

## 五、代码实现详解

### 5.1 Pinecone 配置类

```java
package com.example.pinecone.config;

import org.springframework.ai.embedding.EmbeddingClient;
import org.springframework.ai.vectorstore.PineconeVectorStore;
import org.springframework.ai.vectorstore.PineconeVectorStoreConfig;
import org.springframework.ai.vectorstore.VectorStore;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

@Configuration
public class PineconeConfig {

    @Value("${spring.ai.vectorstore.pinecone.api-key}")
    private String apiKey;

    @Value("${spring.ai.vectorstore.pinecone.environment}")
    private String environment;

    @Value("${spring.ai.vectorstore.pinecone.index-name}")
    private String indexName;

    @Bean
    public VectorStore vectorStore(EmbeddingClient embeddingClient) {
        PineconeVectorStoreConfig config = PineconeVectorStoreConfig.builder()
            .withApiKey(apiKey)
            .withEnvironment(environment)
            .withIndexName(indexName)
            .withNamespace("default") // 可选的命名空间
            .build();
        
        return new PineconeVectorStore(embeddingClient, config);
    }
}
```

### 5.2 向量数据库服务类

```java
package com.example.pinecone.service;

import org.springframework.ai.document.Document;
import org.springframework.ai.vectorstore.SearchRequest;
import org.springframework.ai.vectorstore.VectorStore;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;

import java.util.List;
import java.util.Map;

@Service
public class PineconeService {

    @Autowired
    private VectorStore vectorStore;

    public void addDocuments(List<Document> documents) {
        vectorStore.add(documents);
    }

    public List<Document> similaritySearch(String query, int topK) {
        return vectorStore.similaritySearch(query, topK);
    }

    public List<Document> similaritySearchWithFilter(String query, int topK, Map<String, Object> filter) {
        SearchRequest request = SearchRequest.query(query)
            .withTopK(topK)
            .withSimilarityThreshold(0.0)
            .withFilterExpression(filter);
        return vectorStore.similaritySearch(request);
    }

    public void deleteDocuments(List<String> ids) {
        vectorStore.delete(ids);
    }
}
```

### 5.3 REST API 控制器

```java
package com.example.pinecone.controller;

import org.springframework.ai.document.Document;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;
import com.example.pinecone.service.PineconeService;

import java.util.ArrayList;
import java.util.List;
import java.util.Map;

@RestController
@RequestMapping("/api/pinecone")
public class PineconeController {

    @Autowired
    private PineconeService pineconeService;

    @PostMapping("/documents")
    public String addDocuments(@RequestBody List<Map<String, Object>> docs) {
        List<Document> documents = new ArrayList<>();
        for (Map<String, Object> doc : docs) {
            String content = (String) doc.get("content");
            @SuppressWarnings("unchecked")
            Map<String, Object> metadata = (Map<String, Object>) doc.get("metadata");
            documents.add(new Document(content, metadata));
        }
        pineconeService.addDocuments(documents);
        return "Documents added successfully";
    }

    @GetMapping("/search")
    public List<Document> search(
            @RequestParam String query,
            @RequestParam(defaultValue = "5") int topK) {
        return pineconeService.similaritySearch(query, topK);
    }

    @PostMapping("/search/filter")
    public List<Document> searchWithFilter(
            @RequestParam String query,
            @RequestParam(defaultValue = "5") int topK,
            @RequestBody Map<String, Object> filter) {
        return pineconeService.similaritySearchWithFilter(query, topK, filter);
    }

    @DeleteMapping("/documents")
    public String deleteDocuments(@RequestBody List<String> ids) {
        pineconeService.deleteDocuments(ids);
        return "Documents deleted successfully";
    }
}
```

import org.springframework.web.client.RestTemplate;
import org.springframework.http.HttpEntity;
import org.springframework.http.HttpHeaders;
import org.springframework.http.MediaType;
import java.util.List;
import java.util.Map;

public class PineconeClient {

    private static final String BASE_URL = "http://localhost:8080/api/pinecone";
    private RestTemplate restTemplate = new RestTemplate();

    public void addDocuments(List<Map<String, Object>> docs) {
        HttpHeaders headers = new HttpHeaders();
        headers.setContentType(MediaType.APPLICATION_JSON);
        HttpEntity<List<Map<String, Object>>> request = new HttpEntity<>(docs, headers);

        String url = BASE_URL + "/search?query={query}&topK={topK}";
        return restTemplate.getForObject(url, List.class, query, topK);

## Java 客户端示例

以下是一个独立的 Java 客户端示例，用于调用 Spring AI Pinecone 向量存储服务：

```java
import org.springframework.web.client.RestTemplate;
import org.springframework.http.*;
import java.util.Map;

public class PineconeClient {
    private final RestTemplate restTemplate;
    private final String baseUrl;

    public PineconeClient(String baseUrl) {
        this.baseUrl = baseUrl;
        this.restTemplate = new RestTemplate();
    }

    public Map<String, Object> search(String query) {
        String url = baseUrl + "/pinecone/search?query=" +
                    java.net.URLEncoder.encode(query, java.nio.charset.StandardCharsets.UTF_8);
        return restTemplate.getForObject(url, Map.class);
    }

    public Map<String, Object> addDocument(String text, Map<String, Object> metadata) {
        HttpHeaders headers = new HttpHeaders();
        headers.setContentType(MediaType.APPLICATION_JSON);

        Map<String, Object> body = Map.of(
            "text", text,
            "metadata", metadata
        );
        HttpEntity<Map<String, Object>> request = new HttpEntity<>(body, headers);

        ResponseEntity<Map> response = restTemplate.postForEntity(
            baseUrl + "/pinecone/add", request, Map.class);

        return response.getBody();
    }

    public static void main(String[] args) {
        PineconeClient client = new PineconeClient("http://localhost:8080");

        // 添加文档
        Map<String, Object> metadata = Map.of("category", "database", "type", "vector");
        Map<String, Object> addResult = client.addDocument(
            "Pinecone 是一个托管的向量数据库服务",
            metadata
        );
        System.out.println("Document added: " + addResult.get("id"));

        // 搜索文档
        Map<String, Object> searchResult = client.search("向量数据库");
        System.out.println("\nSearch results: " + searchResult.get("results"));
    }
}
```
