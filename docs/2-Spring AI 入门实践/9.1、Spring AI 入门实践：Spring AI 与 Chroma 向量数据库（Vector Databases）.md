# Spring AI 入门实践：Spring AI 与 Chroma 向量数据库（Vector Databases）

## 什么是 Chroma

Chroma 是一个开源的向量数据库，专为 AI 应用设计，提供简单易用的 API 来存储和检索向量嵌入。

## 准备工作

### 1. 安装 Chroma

可以通过 pip 安装 Chroma：

```bash
pip install chromadb
```

### 2. 添加 Spring AI 依赖

```xml
<dependency>
    <groupId>org.springframework.ai</groupId>
    <artifactId>spring-ai-starter-vector-store-chroma</artifactId>
</dependency>
```

## 配置 Chroma 向量存储

### 1. 配置属性

在 `application.properties` 中添加：

```properties
# Chroma 配置
spring.ai.vectorstore.chroma.url=localhost:8000
spring.ai.vectorstore.chroma.collection-name=my-collection
```

### 2. 创建 Chroma 向量存储 Bean

```java
@Bean
public VectorStore vectorStore(EmbeddingClient embeddingClient) {
    return new ChromaVectorStore(
        embeddingClient,
        "localhost:8000",
        "my-collection"
    );
}
```

## 基本操作

### 1. 添加文档

```java
@Autowired
private VectorStore vectorStore;

public void addDocuments() {
    List<Document> documents = new ArrayList<>();
    documents.add(new Document("Spring Boot 是一个快速开发框架", Map.of("category", "framework")));
    documents.add(new Document("Spring AI 提供了 AI 能力集成", Map.of("category", "ai")));
    documents.add(new Document("Chroma 是一个向量数据库", Map.of("category", "database")));
    
    vectorStore.add(documents);
}
```

### 2. 相似度搜索

```java
public void searchDocuments() {
    List<Document> results = vectorStore.similaritySearch(
        "什么是 Spring 框架？",
        2
    );
    
    for (Document document : results) {
        System.out.println("内容: " + document.getContent());
        System.out.println("相似度: " + document.getMetadata().get("score"));
    }
}
```

### 3. 带过滤条件的搜索

```java
public void searchWithFilter() {
    Map<String, Object> filter = Map.of("category", "ai");
    
    List<Document> results = vectorStore.similaritySearch(
        "AI 相关技术",
        2,
        filter
    );
    
    for (Document document : results) {
        System.out.println("内容: " + document.getContent());
    }
}
```

## 高级特性

### 1. 批量操作

```java
public void batchOperations() {
    // 批量添加
    List<Document> documents = new ArrayList<>();
    for (int i = 0; i < 100; i++) {
        documents.add(new Document("文档 " + i, Map.of("id", i)));
    }
    vectorStore.add(documents);
    
    // 批量删除
    List<String> ids = documents.stream()
        .map(Document::getId)
        .collect(Collectors.toList());
    vectorStore.delete(ids);
}
```

### 2. 自定义嵌入模型

```java
@Bean
public EmbeddingClient embeddingClient() {
    return new OpenAiEmbeddingClient(
        new OpenAiApi(System.getenv("OPENAI_API_KEY"))
    );
}

@Bean
public VectorStore vectorStore() {
    return new ChromaVectorStore(
        embeddingClient(),
        "localhost:8000",
        "my-collection"
    );
}
```

## 示例代码

完整的示例代码如下：

```java
import org.springframework.ai.document.Document;
import org.springframework.ai.embedding.EmbeddingClient;
import org.springframework.ai.vectorstore.ChromaVectorStore;
import org.springframework.ai.vectorstore.VectorStore;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;

import java.util.ArrayList;
import java.util.List;
import java.util.Map;

@Service
public class ChromaService {

    @Autowired
    private VectorStore vectorStore;

    public void demo() {
        // 添加文档
        List<Document> documents = new ArrayList<>();
        documents.add(new Document("Spring Boot 是一个快速开发框架", Map.of("category", "framework")));
        documents.add(new Document("Spring AI 提供了 AI 能力集成", Map.of("category", "ai")));
        documents.add(new Document("Chroma 是一个向量数据库", Map.of("category", "database")));
        documents.add(new Document("向量数据库用于存储和检索嵌入向量", Map.of("category", "database")));
        documents.add(new Document("RAG 技术结合了检索和生成", Map.of("category", "ai")));
        
        vectorStore.add(documents);
        System.out.println("文档添加完成");
        
        // 相似度搜索
        System.out.println("\n相似度搜索结果：");
        List<Document> results = vectorStore.similaritySearch("Spring 相关技术", 3);
        for (int i = 0; i < results.size(); i++) {
            Document doc = results.get(i);
            System.out.println((i + 1) + ". " + doc.getContent());
            System.out.println("   类别: " + doc.getMetadata().get("category"));
        }
        
        // 带过滤条件的搜索
        System.out.println("\n带过滤条件的搜索结果：");
        Map<String, Object> filter = Map.of("category", "ai");
        results = vectorStore.similaritySearch("技术", 2, filter);
        for (int i = 0; i < results.size(); i++) {
            Document doc = results.get(i);
            System.out.println((i + 1) + ". " + doc.getContent());
        }
    }
}
```

## 总结

Chroma 向量数据库提供了简单易用的 API，与 Spring AI 集成后，可以轻松实现文档的向量存储和相似度搜索。这为 RAG 等 AI 应用提供了强大的支持。
