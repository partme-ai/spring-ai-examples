# Spring AI 入门实践：Spring AI 与 Neo4j 向量数据库（Vector Databases）

## 什么是 Neo4j

Neo4j 是一个图形数据库，同时也支持向量搜索功能，适用于需要存储和查询复杂关系数据的场景。

## 准备工作

### 1. 安装 Neo4j

可以使用 Docker 快速启动 Neo4j：

```bash
docker run -d --name neo4j -p 7474:7474 -p 7687:7687 -e NEO4J_AUTH=neo4j/password neo4j:5.15
```

### 2. 添加 Spring AI 依赖

```xml
<dependency>
    <groupId>org.springframework.ai</groupId>
    <artifactId>spring-ai-starter-vector-store-neo4j</artifactId>
</dependency>
```

## 配置 Neo4j 向量存储

### 1. 配置属性

在 `application.properties` 中添加：

```properties
# Neo4j 配置
spring.neo4j.uri=bolt://localhost:7687
spring.neo4j.authentication.username=neo4j
spring.neo4j.authentication.password=password

# Spring AI 配置
spring.ai.vectorstore.neo4j.index-name=vector-index
spring.ai.vectorstore.neo4j.node-label=Document
spring.ai.vectorstore.neo4j.embedding-property=embedding
spring.ai.vectorstore.neo4j.content-property=content
```

### 2. 创建 Neo4j 向量存储 Bean

```java
@Bean
public VectorStore vectorStore(EmbeddingClient embeddingClient, Neo4jClient neo4jClient) {
    return new Neo4jVectorStore(
        embeddingClient,
        neo4jClient,
        "vector-index",
        "Document",
        "embedding",
        "content"
    );
}
```

## 基本操作

### 1. 添加文档

```java
@Autowired
private VectorStore vectorStore;

public void addDocuments() {
    List<Document> documents = new ArrayList<>();
    documents.add(new Document("Spring Boot 是一个快速开发框架", Map.of("category", "framework")));
    documents.add(new Document("Spring AI 提供了 AI 能力集成", Map.of("category", "ai")));
    documents.add(new Document("Neo4j 是一个图形数据库", Map.of("category", "database")));
    
    vectorStore.add(documents);
}
```

### 2. 相似度搜索

```java
public void searchDocuments() {
    List<Document> results = vectorStore.similaritySearch(
        "什么是 Spring 框架？",
        2
    );
    
    for (Document document : results) {
        System.out.println("内容: " + document.getContent());
        System.out.println("相似度: " + document.getMetadata().get("score"));
    }
}
```

### 3. 带过滤条件的搜索

```java
public void searchWithFilter() {
    Map<String, Object> filter = Map.of("category", "ai");
    
    List<Document> results = vectorStore.similaritySearch(
        "AI 相关技术",
        2,
        filter
    );
    
    for (Document document : results) {
        System.out.println("内容: " + document.getContent());
    }
}
```

## 高级特性

### 1. 图形关系查询

Neo4j 的优势在于可以存储和查询复杂的关系。我们可以利用这一特性来构建更加丰富的知识图谱：

```java
public void createRelationships() {
    // 使用 Neo4jClient 创建关系
    neo4jClient.query("MATCH (d1:Document {content: $content1}), (d2:Document {content: $content2}) CREATE (d1)-[:RELATED_TO]->(d2)")
        .bind("Spring Boot 是一个快速开发框架").to("content1")
        .bind("Spring AI 提供了 AI 能力集成").to("content2")
        .execute();
}
```

### 2. 自定义索引配置

```java
@Bean
public VectorStore vectorStore(EmbeddingClient embeddingClient, Neo4jClient neo4jClient) {
    Neo4jVectorStoreConfig config = Neo4jVectorStoreConfig.builder()
        .withIndexName("vector-index")
        .withNodeLabel("Document")
        .withEmbeddingProperty("embedding")
        .withContentProperty("content")
        .withSimilarityFunction("cosine")
        .build();
    
    return new Neo4jVectorStore(embeddingClient, neo4jClient, config);
}
```

## 示例代码

完整的示例代码如下：

```java
import org.springframework.ai.document.Document;
import org.springframework.ai.embedding.EmbeddingClient;
import org.springframework.ai.vectorstore.Neo4jVectorStore;
import org.springframework.ai.vectorstore.VectorStore;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.data.neo4j.core.Neo4jClient;
import org.springframework.stereotype.Service;

import java.util.ArrayList;
import java.util.List;
import java.util.Map;

@Service
public class Neo4jService {

    @Autowired
    private VectorStore vectorStore;

    @Autowired
    private Neo4jClient neo4jClient;

    public void demo() {
        // 添加文档
        List<Document> documents = new ArrayList<>();
        documents.add(new Document("Spring Boot 是一个快速开发框架", Map.of("category", "framework")));
        documents.add(new Document("Spring AI 提供了 AI 能力集成", Map.of("category", "ai")));
        documents.add(new Document("Neo4j 是一个图形数据库", Map.of("category", "database")));
        documents.add(new Document("向量数据库用于存储和检索嵌入向量", Map.of("category", "database")));
        documents.add(new Document("RAG 技术结合了检索和生成", Map.of("category", "ai")));
        
        vectorStore.add(documents);
        System.out.println("文档添加完成");
        
        // 创建关系
        createRelationships();
        System.out.println("关系创建完成");
        
        // 相似度搜索
        System.out.println("\n相似度搜索结果：");
        List<Document> results = vectorStore.similaritySearch("Spring 相关技术", 3);
        for (int i = 0; i < results.size(); i++) {
            Document doc = results.get(i);
            System.out.println((i + 1) + ". " + doc.getContent());
            System.out.println("   类别: " + doc.getMetadata().get("category"));
        }
        
        // 带过滤条件的搜索
        System.out.println("\n带过滤条件的搜索结果：");
        Map<String, Object> filter = Map.of("category", "ai");
        results = vectorStore.similaritySearch("技术", 2, filter);
        for (int i = 0; i < results.size(); i++) {
            Document doc = results.get(i);
            System.out.println((i + 1) + ". " + doc.getContent());
        }
    }

    private void createRelationships() {
        // 创建文档之间的关系
        neo4jClient.query("MATCH (d1:Document {content: $content1}), (d2:Document {content: $content2}) CREATE (d1)-[:RELATED_TO]->(d2)")
            .bind("Spring Boot 是一个快速开发框架").to("content1")
            .bind("Spring AI 提供了 AI 能力集成").to("content2")
            .execute();

        neo4jClient.query("MATCH (d1:Document {content: $content1}), (d2:Document {content: $content2}) CREATE (d1)-[:RELATED_TO]->(d2)")
            .bind("Spring AI 提供了 AI 能力集成").to("content1")
            .bind("RAG 技术结合了检索和生成").to("content2")
            .execute();

        neo4jClient.query("MATCH (d1:Document {content: $content1}), (d2:Document {content: $content2}) CREATE (d1)-[:RELATED_TO]->(d2)")
            .bind("Neo4j 是一个图形数据库").to("content1")
            .bind("向量数据库用于存储和检索嵌入向量").to("content2")
            .execute();
    }
}
```

## 总结

Neo4j 向量数据库结合了图形数据库和向量搜索的优势，与 Spring AI 集成后，可以构建更加复杂的知识图谱和智能检索系统。这为需要处理复杂关系和语义相似性的 AI 应用提供了强大的支持。
