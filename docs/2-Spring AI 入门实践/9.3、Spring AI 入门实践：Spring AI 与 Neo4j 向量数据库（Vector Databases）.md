# Spring AI 入门实践：Spring AI 与 Neo4j 向量数据库（Vector Databases）

> 基于 Spring AI 框架实现与 Neo4j 向量数据库的集成，结合图形数据库和向量搜索的优势，提供文档向量存储、相似度搜索、知识图谱关系查询等功能。

---

## 一、项目概述

### 1.1 项目定位

本项目是 Spring AI 框架下集成 Neo4j 向量数据库的示例，展示了如何在 Java/Spring Boot 应用中结合图形数据库和向量搜索，构建知识图谱和智能检索系统。

### 1.2 技术栈

| 组件 | 版本 | 说明 |
|------|------|------|
| Spring Boot | 3.5.6 | 基础框架 |
| Spring AI | 1.1.4 | AI 能力集成 |
| Neo4j | 5.15+ | 图形数据库 |
| spring-ai-starter-vector-store-neo4j | - | Spring AI Neo4j 集成 |

### 1.3 核心功能

- ✅ 文档向量存储：支持将文档转换为向量并存储
- ✅ 相似度搜索：基于向量相似度检索相关文档
- ✅ 图形关系查询：支持节点和关系的图形查询
- ✅ 元数据过滤：支持基于元数据的过滤查询
- ✅ 知识图谱构建：结合向量搜索和图形关系

## 二、Neo4j 向量数据库简介

### 2.1 Neo4j 介绍

Neo4j 是一个图形数据库，同时也支持向量搜索功能，适用于需要存储和查询复杂关系数据的场景。Neo4j 结合了图形数据库的关系查询能力和向量搜索的语义检索能力，为知识图谱应用提供强大支持。

### 2.2 核心特性

| 特性 | 说明 |
|------|------|
| **图形数据库** | 原生支持节点和关系的存储和查询 |
| **Cypher 查询语言** | 强大的图形查询语言 |
| **向量搜索** | 支持向量相似度搜索 |
| **ACID 事务** | 支持完整的事务特性 |
| **高可用性** | 支持集群部署和数据复制 |
| **灵活 Schema** | 无需预定义 schema，灵活存储数据 |

### 2.3 应用场景

- 知识图谱构建和查询
- 推荐系统
- 社交网络分析
- 语义搜索和问答
- 欺诈检测
- 网络和 IT 运营

## 五、代码实现详解

### 5.1 Neo4j 配置类

```java
package com.example.neo4j.config;

import org.springframework.ai.embedding.EmbeddingClient;
import org.springframework.ai.vectorstore.Neo4jVectorStore;
import org.springframework.ai.vectorstore.VectorStore;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.data.neo4j.core.Neo4jClient;

@Configuration
public class Neo4jConfig {

    @Value("${spring.ai.vectorstore.neo4j.index-name:vector-index}")
    private String indexName;

    @Value("${spring.ai.vectorstore.neo4j.node-label:Document}")
    private String nodeLabel;

    @Value("${spring.ai.vectorstore.neo4j.embedding-property:embedding}")
    private String embeddingProperty;

    @Value("${spring.ai.vectorstore.neo4j.content-property:content}")
    private String contentProperty;

    @Bean
    public VectorStore vectorStore(EmbeddingClient embeddingClient, Neo4jClient neo4jClient) {
        return new Neo4jVectorStore(
            embeddingClient,
            neo4jClient,
            indexName,
            nodeLabel,
            embeddingProperty,
            contentProperty
        );
    }
}
```

### 5.2 向量数据库服务类

```java
package com.example.neo4j.service;

import org.springframework.ai.document.Document;
import org.springframework.ai.vectorstore.SearchRequest;
import org.springframework.ai.vectorstore.VectorStore;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.data.neo4j.core.Neo4jClient;
import org.springframework.stereotype.Service;

import java.util.List;
import java.util.Map;

@Service
public class Neo4jService {

    @Autowired
    private VectorStore vectorStore;

    @Autowired
    private Neo4jClient neo4jClient;

    public void addDocuments(List<Document> documents) {
        vectorStore.add(documents);
    }

    public List<Document> similaritySearch(String query, int topK) {
        return vectorStore.similaritySearch(query, topK);
    }

    public List<Document> similaritySearchWithFilter(String query, int topK, Map<String, Object> filter) {
        SearchRequest request = SearchRequest.query(query)
            .withTopK(topK)
            .withSimilarityThreshold(0.0)
            .withFilterExpression(filter);
        return vectorStore.similaritySearch(request);
    }

    public void deleteDocuments(List<String> ids) {
        vectorStore.delete(ids);
    }

    public void createRelationships(String content1, String content2, String relationshipType) {
        neo4jClient.query("""
                MATCH (d1:Document {content: $content1}), (d2:Document {content: $content2})
                CREATE (d1)-[r:""" + relationshipType + """]->(d2)
                RETURN r
                """)
            .bind(content1).to("content1")
            .bind(content2).to("content2")
            .execute();
    }
}
```

### 5.3 REST API 控制器

```java
package com.example.neo4j.controller;

import org.springframework.ai.document.Document;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;
import com.example.neo4j.service.Neo4jService;

import java.util.ArrayList;
import java.util.List;
import java.util.Map;

@RestController
@RequestMapping("/api/neo4j")
public class Neo4jController {

    @Autowired
    private Neo4jService neo4jService;

    @PostMapping("/documents")
    public String addDocuments(@RequestBody List<Map<String, Object>> docs) {
        List<Document> documents = new ArrayList<>();
        for (Map<String, Object> doc : docs) {
            String content = (String) doc.get("content");
            @SuppressWarnings("unchecked")
            Map<String, Object> metadata = (Map<String, Object>) doc.get("metadata");
            documents.add(new Document(content, metadata));
        }
        neo4jService.addDocuments(documents);
        return "Documents added successfully";
    }

    @GetMapping("/search")
    public List<Document> search(
            @RequestParam String query,
            @RequestParam(defaultValue = "5") int topK) {
        return neo4jService.similaritySearch(query, topK);
    }

    @PostMapping("/search/filter")
    public List<Document> searchWithFilter(
            @RequestParam String query,
            @RequestParam(defaultValue = "5") int topK,
            @RequestBody Map<String, Object> filter) {
        return neo4jService.similaritySearchWithFilter(query, topK, filter);
    }

    @DeleteMapping("/documents")
    public String deleteDocuments(@RequestBody List<String> ids) {
        neo4jService.deleteDocuments(ids);
        return "Documents deleted successfully";
    }

    @PostMapping("/relationships")
    public String createRelationship(@RequestBody Map<String, String> request) {
        String content1 = request.get("content1");
        String content2 = request.get("content2");
        String relationshipType = request.getOrDefault("relationshipType", "RELATED_TO");
        neo4jService.createRelationships(content1, content2, relationshipType);
        return "Relationship created successfully";
    }
}
```

### 8.2 Java 客户端

```java
import org.springframework.web.client.RestTemplate;
import org.springframework.http.HttpEntity;
import org.springframework.http.HttpHeaders;
import org.springframework.http.MediaType;
import java.util.List;
import java.util.Map;

public class Neo4jClient {

    private static final String BASE_URL = "http://localhost:8080/api/neo4j";
    private RestTemplate restTemplate = new RestTemplate();

    public void addDocuments(List<Map<String, Object>> docs) {
        HttpHeaders headers = new HttpHeaders();
        headers.setContentType(MediaType.APPLICATION_JSON);
        HttpEntity<List<Map<String, Object>>> request = new HttpEntity<>(docs, headers);
        restTemplate.postForObject(BASE_URL + "/documents", request, String.class);
    }

    public List<Map<String, Object>> search(String query, int topK) {
        String url = BASE_URL + "/search?query={query}&topK={topK}";
        return restTemplate.getForObject(url, List.class, query, topK);
    }
}
```

## 十、常见问题

### Q1: Neo4j 服务连接失败？

检查 Neo4j 服务是否正常启动，确认配置的 URI 是否正确。可以通过访问 http://localhost:7474 验证服务状态。

### Q2: 如何使用 Cypher 查询图形关系？

可以使用 Neo4jClient 执行自定义 Cypher 查询，也可以通过 Neo4j 浏览器进行交互式查询。

### Q3: 向量索引如何优化？

Neo4j 的向量索引会自动管理，但可以通过调整索引参数来优化性能。参考 Neo4j 官方文档了解详细配置。
