# Spring AI 入门实践：Spring AI 与 Weaviate 向量数据库（Vector Databases）

> 基于 Spring AI 框架实现与 Weaviate 向量数据库的集成，使用开源的向量搜索引擎，提供向量搜索、知识图谱和实体链接功能。

---

## 一、项目概述

### 1.1 项目定位

本项目是 Spring AI 框架下集成 Weaviate 向量数据库的示例，展示了如何在 Java/Spring Boot 应用中使用开源的 Weaviate 向量搜索引擎，结合向量搜索和知识图谱功能，构建 RAG 应用。

### 1.2 技术栈

| 组件 | 版本 | 说明 |
|------|------|------|
| Spring Boot | 3.5.6 | 基础框架 |
| Spring AI | 1.1.4 | AI 能力集成 |
| Weaviate | - | 开源向量搜索引擎 |
| spring-ai-starter-vector-store-weaviate | - | Spring AI Weaviate 集成 |

### 1.3 核心功能

- ✅ 向量搜索：支持高效的向量相似度搜索
- ✅ 知识图谱：内置知识图谱和实体链接功能
- ✅ 模块架构：支持多种向量器和模块
- ✅ 元数据过滤：强大的元数据过滤查询
- ✅ REST API：简单易用的 GraphQL 和 REST 接口
- ✅ 分布式部署：支持集群模式
- ✅ 开源免费：BSD 3-Clause 许可证

## 二、Weaviate 向量数据库简介

### 2.1 Weaviate 介绍

Weaviate 是一个开源的向量搜索引擎，专为 AI 应用设计，提供向量相似度搜索和知识图谱功能。Weaviate 支持多种向量器模块，可以轻松集成不同的嵌入模型，同时提供实体链接和知识图谱功能。

### 2.2 核心特性

| 特性 | 说明 |
|------|------|
| **向量搜索** | 支持高效的向量相似度搜索 |
| **知识图谱** | 内置知识图谱和实体链接功能 |
| **模块架构** | 支持多种向量器和模块 |
| **元数据过滤** | 强大的元数据过滤查询 |
| **REST API** | 简单易用的 GraphQL 和 REST 接口 |
| **分布式部署** | 支持集群模式 |
| **开源免费** | BSD 3-Clause 许可证 |

### 2.3 向量器模块

Weaviate 支持多种向量器模块：

| 向量器 | 说明 |
|--------|------|
| **text2vec-openai** | 使用 OpenAI 嵌入模型 |
| **text2vec-cohere** | 使用 Cohere 嵌入模型 |
| **text2vec-huggingface** | 使用 Hugging Face 模型 |
| **text2vec-contextionary** | Weaviate 内置向量器 |
| **multi2vec-clip** | 多模态向量器（图像+文本） |

## 五、代码实现详解

### 5.1 Weaviate 配置类

```java
package com.example.weaviate.config;

import org.springframework.ai.embedding.EmbeddingClient;
import org.springframework.ai.vectorstore.VectorStore;
import org.springframework.ai.vectorstore.WeaviateVectorStore;
import org.springframework.ai.vectorstore.WeaviateVectorStoreConfig;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

@Configuration
public class WeaviateConfig {

    @Value("${spring.ai.vectorstore.weaviate.url}")
    private String url;

    @Value("${spring.ai.vectorstore.weaviate.collection-name}")
    private String collectionName;

    @Bean
    public VectorStore vectorStore(EmbeddingClient embeddingClient) {
        WeaviateVectorStoreConfig config = WeaviateVectorStoreConfig.builder()
            .withUrl(url)
            .withCollectionName(collectionName)
            .withVectorizer("text2vec-openai")
            .build();
        
        return new WeaviateVectorStore(embeddingClient, config);
    }
}
```

### 5.2 向量数据库服务类

```java
package com.example.weaviate.service;

import org.springframework.ai.document.Document;
import org.springframework.ai.vectorstore.SearchRequest;
import org.springframework.ai.vectorstore.VectorStore;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;

import java.util.List;
import java.util.Map;

@Service
public class WeaviateService {

    @Autowired
    private VectorStore vectorStore;

    public void addDocuments(List<Document> documents) {
        vectorStore.add(documents);
    }

    public List<Document> similaritySearch(String query, int topK) {
        return vectorStore.similaritySearch(query, topK);
    }

    public List<Document> similaritySearchWithFilter(String query, int topK, Map<String, Object> filter) {
        SearchRequest request = SearchRequest.query(query)
            .withTopK(topK)
            .withSimilarityThreshold(0.0)
            .withFilterExpression(filter);
        return vectorStore.similaritySearch(request);
    }

    public void deleteDocuments(List<String> ids) {
        vectorStore.delete(ids);
    }
}
```

### 5.3 REST API 控制器

```java
package com.example.weaviate.controller;

import org.springframework.ai.document.Document;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;
import com.example.weaviate.service.WeaviateService;

import java.util.ArrayList;
import java.util.List;
import java.util.Map;

@RestController
@RequestMapping("/api/weaviate")
public class WeaviateController {

    @Autowired
    private WeaviateService weaviateService;

    @PostMapping("/documents")
    public String addDocuments(@RequestBody List<Map<String, Object>> docs) {
        List<Document> documents = new ArrayList<>();
        for (Map<String, Object> doc : docs) {
            String content = (String) doc.get("content");
            @SuppressWarnings("unchecked")
            Map<String, Object> metadata = (Map<String, Object>) doc.get("metadata");
            documents.add(new Document(content, metadata));
        }
        weaviateService.addDocuments(documents);
        return "Documents added successfully";
    }

    @GetMapping("/search")
    public List<Document> search(
            @RequestParam String query,
            @RequestParam(defaultValue = "5") int topK) {
        return weaviateService.similaritySearch(query, topK);
    }

    @PostMapping("/search/filter")
    public List<Document> searchWithFilter(
            @RequestParam String query,
            @RequestParam(defaultValue = "5") int topK,
            @RequestBody Map<String, Object> filter) {
        return weaviateService.similaritySearchWithFilter(query, topK, filter);
    }

    @DeleteMapping("/documents")
    public String deleteDocuments(@RequestBody List<String> ids) {
        weaviateService.deleteDocuments(ids);
        return "Documents deleted successfully";
    }
}
```

## 十、常见问题

### Q1: 如何选择合适的向量器？

Weaviate 支持多种向量器，选择建议：
- **text2vec-openai**：使用 OpenAI 嵌入模型，质量高
- **text2vec-huggingface**：使用 Hugging Face 模型，可以自定义
- **text2vec-contextionary**：Weaviate 内置向量器，速度快
- **multi2vec-clip**：多模态向量器，支持图像和文本

### Q2: Weaviate 的知识图谱功能如何使用？

Weaviate 的知识图谱功能通过实体链接实现，可以使用 `ref2vec-centroid` 等模块来处理实体关系。详细使用请参考 Weaviate 官方文档。

### Q3: 如何配置 Weaviate 集群？

Weaviate 支持分布式部署，可以通过配置多个节点来实现高可用性和水平扩展。详细配置请参考 Weaviate 官方文档的集群部署部分。
