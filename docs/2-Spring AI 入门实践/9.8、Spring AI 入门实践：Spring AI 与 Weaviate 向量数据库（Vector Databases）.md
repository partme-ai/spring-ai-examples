# Spring AI 入门实践：Spring AI 与 Weaviate 向量数据库（Vector Databases）

## 什么是 Weaviate

Weaviate 是一个开源的向量搜索引擎，专为 AI 应用设计，提供向量相似度搜索和知识图谱功能。

## 准备工作

### 1. 安装 Weaviate

可以使用 Docker 快速启动 Weaviate：

```bash
docker run -d --name weaviate -p 8080:8080 semitechnologies/weaviate:1.24.0
```

### 2. 添加 Spring AI 依赖

```xml
<dependency>
    <groupId>org.springframework.ai</groupId>
    <artifactId>spring-ai-weaviate-store</artifactId>
</dependency>
```

## 配置 Weaviate 向量存储

### 1. 配置属性

在 `application.properties` 中添加：

```properties
# Weaviate 配置
spring.ai.vectorstore.weaviate.url=http://localhost:8080
spring.ai.vectorstore.weaviate.collection-name=Document
```

### 2. 创建 Weaviate 向量存储 Bean

```java
@Bean
public VectorStore vectorStore(EmbeddingClient embeddingClient) {
    return new WeaviateVectorStore(
        embeddingClient,
        "http://localhost:8080",
        "Document"
    );
}
```

## 基本操作

### 1. 添加文档

```java
@Autowired
private VectorStore vectorStore;

public void addDocuments() {
    List<Document> documents = new ArrayList<>();
    documents.add(new Document("Spring Boot 是一个快速开发框架", Map.of("category", "framework")));
    documents.add(new Document("Spring AI 提供了 AI 能力集成", Map.of("category", "ai")));
    documents.add(new Document("Weaviate 是一个向量搜索引擎", Map.of("category", "database")));
    
    vectorStore.add(documents);
}
```

### 2. 相似度搜索

```java
public void searchDocuments() {
    List<Document> results = vectorStore.similaritySearch(
        "什么是 Spring 框架？",
        2
    );
    
    for (Document document : results) {
        System.out.println("内容: " + document.getContent());
        System.out.println("相似度: " + document.getMetadata().get("score"));
    }
}
```

### 3. 带过滤条件的搜索

```java
public void searchWithFilter() {
    Map<String, Object> filter = Map.of("category", "ai");
    
    List<Document> results = vectorStore.similaritySearch(
        "AI 相关技术",
        2,
        filter
    );
    
    for (Document document : results) {
        System.out.println("内容: " + document.getContent());
    }
}
```

## 高级特性

### 1. 自定义索引配置

```java
@Bean
public VectorStore vectorStore(EmbeddingClient embeddingClient) {
    WeaviateVectorStoreConfig config = WeaviateVectorStoreConfig.builder()
        .withUrl("http://localhost:8080")
        .withCollectionName("Document")
        .withVectorizer("text2vec-openai") // 使用 OpenAI 向量器
        .build();
    
    return new WeaviateVectorStore(embeddingClient, config);
}
```

### 2. 批量操作

```java
public void batchOperations() {
    // 批量添加
    List<Document> documents = new ArrayList<>();
    for (int i = 0; i < 100; i++) {
        documents.add(new Document("文档 " + i, Map.of("id", i, "category", "test")));
    }
    vectorStore.add(documents);
    
    // 批量删除
    List<String> ids = documents.stream()
        .map(Document::getId)
        .collect(Collectors.toList());
    vectorStore.delete(ids);
}
```

## 示例代码

完整的示例代码如下：

```java
import org.springframework.ai.document.Document;
import org.springframework.ai.embedding.EmbeddingClient;
import org.springframework.ai.vectorstore.WeaviateVectorStore;
import org.springframework.ai.vectorstore.VectorStore;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;

import java.util.ArrayList;
import java.util.List;
import java.util.Map;

@Service
public class WeaviateService {

    @Autowired
    private VectorStore vectorStore;

    public void demo() {
        // 添加文档
        List<Document> documents = new ArrayList<>();
        documents.add(new Document("Spring Boot 是一个快速开发框架", Map.of("category", "framework")));
        documents.add(new Document("Spring AI 提供了 AI 能力集成", Map.of("category", "ai")));
        documents.add(new Document("Weaviate 是一个向量搜索引擎", Map.of("category", "database")));
        documents.add(new Document("向量数据库用于存储和检索嵌入向量", Map.of("category", "database")));
        documents.add(new Document("RAG 技术结合了检索和生成", Map.of("category", "ai")));
        
        vectorStore.add(documents);
        System.out.println("文档添加完成");
        
        // 相似度搜索
        System.out.println("\n相似度搜索结果：");
        List<Document> results = vectorStore.similaritySearch("Spring 相关技术", 3);
        for (int i = 0; i < results.size(); i++) {
            Document doc = results.get(i);
            System.out.println((i + 1) + ". " + doc.getContent());
            System.out.println("   类别: " + doc.getMetadata().get("category"));
        }
        
        // 带过滤条件的搜索
        System.out.println("\n带过滤条件的搜索结果：");
        Map<String, Object> filter = Map.of("category", "ai");
        results = vectorStore.similaritySearch("技术", 2, filter);
        for (int i = 0; i < results.size(); i++) {
            Document doc = results.get(i);
            System.out.println((i + 1) + ". " + doc.getContent());
        }
    }
}
```

## 总结

Weaviate 向量数据库提供了强大的向量搜索和知识图谱功能，与 Spring AI 集成后，可以构建更加智能的检索系统。这为需要处理复杂语义关系的 AI 应用提供了强大的支持。
