# Spring AI 入门实践：Spring AI 与 PGvector 向量数据库（Vector Databases）

## 什么是 PGvector

PGvector 是 PostgreSQL 数据库的一个扩展，用于存储和搜索向量数据，为 PostgreSQL 添加了向量相似度搜索能力。

## 准备工作

### 1. 安装 PostgreSQL 和 PGvector

可以使用 Docker 快速启动带有 PGvector 扩展的 PostgreSQL：

```bash
docker run -d --name postgres -p 5432:5432 -e POSTGRES_PASSWORD=password -e POSTGRES_USER=postgres -e POSTGRES_DB=vectordb ankane/pgvector
```

### 2. 添加 Spring AI 依赖

```xml
<dependency>
    <groupId>org.springframework.ai</groupId>
    <artifactId>spring-ai-starter-vector-store-pgvector</artifactId>
</dependency>
<dependency>
    <groupId>org.postgresql</groupId>
    <artifactId>postgresql</artifactId>
</dependency>
```

## 配置 PGvector 向量存储

### 1. 配置属性

在 `application.properties` 中添加：

```properties
# PostgreSQL 配置
spring.datasource.url=jdbc:postgresql://localhost:5432/vectordb
spring.datasource.username=postgres
spring.datasource.password=password

# Spring AI 配置
spring.ai.vectorstore.pgvector.table-name=documents
spring.ai.vectorstore.pgvector.vector-column=embedding
spring.ai.vectorstore.pgvector.content-column=content
spring.ai.vectorstore.pgvector.metadata-column=metadata
```

### 2. 创建 PGvector 向量存储 Bean

```java
@Bean
public VectorStore vectorStore(EmbeddingClient embeddingClient, JdbcTemplate jdbcTemplate) {
    return new PgVectorStore(
        embeddingClient,
        jdbcTemplate,
        "documents",
        "embedding",
        "content",
        "metadata"
    );
}
```

## 基本操作

### 1. 添加文档

```java
@Autowired
private VectorStore vectorStore;

public void addDocuments() {
    List<Document> documents = new ArrayList<>();
    documents.add(new Document("Spring Boot 是一个快速开发框架", Map.of("category", "framework")));
    documents.add(new Document("Spring AI 提供了 AI 能力集成", Map.of("category", "ai")));
    documents.add(new Document("PGvector 是 PostgreSQL 的向量扩展", Map.of("category", "database")));
    
    vectorStore.add(documents);
}
```

### 2. 相似度搜索

```java
public void searchDocuments() {
    List<Document> results = vectorStore.similaritySearch(
        "什么是 Spring 框架？",
        2
    );
    
    for (Document document : results) {
        System.out.println("内容: " + document.getContent());
        System.out.println("相似度: " + document.getMetadata().get("score"));
    }
}
```

### 3. 带过滤条件的搜索

```java
public void searchWithFilter() {
    Map<String, Object> filter = Map.of("category", "ai");
    
    List<Document> results = vectorStore.similaritySearch(
        "AI 相关技术",
        2,
        filter
    );
    
    for (Document document : results) {
        System.out.println("内容: " + document.getContent());
    }
}
```

## 高级特性

### 1. 自定义表结构

可以自定义表结构，添加额外的列来存储更多信息：

```sql
CREATE TABLE documents (
    id SERIAL PRIMARY KEY,
    content TEXT,
    embedding VECTOR(1536),
    metadata JSONB,
    category TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 创建向量索引
CREATE INDEX ON documents USING ivfflat (embedding vector_cosine_ops) WITH (lists = 100);
```

### 2. 批量操作

```java
public void batchOperations() {
    // 批量添加
    List<Document> documents = new ArrayList<>();
    for (int i = 0; i < 100; i++) {
        documents.add(new Document("文档 " + i, Map.of("id", i, "category", "test")));
    }
    vectorStore.add(documents);
    
    // 批量删除
    List<String> ids = documents.stream()
        .map(Document::getId)
        .collect(Collectors.toList());
    vectorStore.delete(ids);
}
```

## 示例代码

完整的示例代码如下：

```java
import org.springframework.ai.document.Document;
import org.springframework.ai.embedding.EmbeddingClient;
import org.springframework.ai.vectorstore.PgVectorStore;
import org.springframework.ai.vectorstore.VectorStore;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.jdbc.core.JdbcTemplate;
import org.springframework.stereotype.Service;

import java.util.ArrayList;
import java.util.List;
import java.util.Map;

@Service
public class PgVectorService {

    @Autowired
    private VectorStore vectorStore;

    public void demo() {
        // 添加文档
        List<Document> documents = new ArrayList<>();
        documents.add(new Document("Spring Boot 是一个快速开发框架", Map.of("category", "framework")));
        documents.add(new Document("Spring AI 提供了 AI 能力集成", Map.of("category", "ai")));
        documents.add(new Document("PGvector 是 PostgreSQL 的向量扩展", Map.of("category", "database")));
        documents.add(new Document("向量数据库用于存储和检索嵌入向量", Map.of("category", "database")));
        documents.add(new Document("RAG 技术结合了检索和生成", Map.of("category", "ai")));
        
        vectorStore.add(documents);
        System.out.println("文档添加完成");
        
        // 相似度搜索
        System.out.println("\n相似度搜索结果：");
        List<Document> results = vectorStore.similaritySearch("Spring 相关技术", 3);
        for (int i = 0; i < results.size(); i++) {
            Document doc = results.get(i);
            System.out.println((i + 1) + ". " + doc.getContent());
            System.out.println("   类别: " + doc.getMetadata().get("category"));
        }
        
        // 带过滤条件的搜索
        System.out.println("\n带过滤条件的搜索结果：");
        Map<String, Object> filter = Map.of("category", "ai");
        results = vectorStore.similaritySearch("技术", 2, filter);
        for (int i = 0; i < results.size(); i++) {
            Document doc = results.get(i);
            System.out.println((i + 1) + ". " + doc.getContent());
        }
    }
}
```

## 总结

PGvector 为 PostgreSQL 数据库添加了向量搜索能力，与 Spring AI 集成后，可以利用 PostgreSQL 的强大功能来存储和检索向量数据。这为需要结合关系型数据库和向量搜索的应用提供了理想的解决方案。
