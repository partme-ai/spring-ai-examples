# Spring AI 入门实践：Spring AI 与 Redis Vector Search（Vector Databases）

> 基于 Spring AI 框架实现与 Redis Vector Search 的集成，利用 Redis 的高性能内存数据存储，提供低延迟的向量搜索和元数据过滤。

---

## 一、项目概述

### 1.1 项目定位

本项目是 Spring AI 框架下集成 Redis Vector Search 的示例，展示了如何在 Java/Spring Boot 应用中利用 Redis 的高性能内存数据存储和向量搜索能力，构建 RAG 应用。

### 1.2 技术栈

| 组件 | 版本 | 说明 |
|------|------|------|
| Spring Boot | 3.5.6 | 基础框架 |
| Spring AI | 1.1.4 | AI 能力集成 |
| Redis | - | 内存数据存储 |
| Redis Vector Search | - | Redis 向量搜索模块 |
| spring-ai-starter-vector-store-redis | - | Spring AI Redis 集成 |

### 1.3 代码地址
**GitHub**：https://github.com/partme-ai/spring-ai-examples/tree/main/spring-ai-ollama-rag-redis
**本地路径**：`spring-ai-ollama-rag-redis/`

### 1.4 核心功能

- ✅ 高性能：利用 Redis 的内存存储特性，提供极低延迟
- ✅ 向量搜索：Redis Vector Search 模块支持
- ✅ 元数据过滤：支持基于元数据的过滤查询
- ✅ 键值存储：结合 Redis 的键值存储能力

- ✅ 集群：支持 Redis 集群模式

## 二、Redis Vector Search 简介

### 2.1 Redis Vector Search 介绍

Redis Vector Search 是 Redis 的一个模块，用于存储和搜索向量数据，为 Redis 添加了向量相似度搜索能力。Redis 作为高性能的内存数据存储，结合向量搜索功能，提供极低的延迟和高吞吐量。

### 2.2 核心特性

| 特性 | 说明 |
|------|------|
| **高性能** | 内存存储，提供极低延迟 |
| **向量搜索** | Redis Vector Search 模块支持 |
| **元数据过滤** | 支持基于元数据的过滤查询 |
| **键值存储** | 结合 Redis 的键值存储能力 |
| **持久化** | 支持 RDB 和 AOF 持久化 |
| **集群** | 支持 Redis 集群模式 |
| **低延迟** | 内存存储提供毫秒级响应 |
| **开源免费** | BSD 许可证 |

### 2.3 Redis Stack

Redis Vector Search 是 Redis Stack 的一部分，Redis Stack 包含了多个 Redis 模块：

| 模块 | 说明 |
|------|------|
| **RediSearch** | 全文搜索和向量搜索 |
| **RedisJSON** | JSON 数据类型支持 |
| **RedisTimeSeries** | 时间序列数据支持 |
| **RedisBloom** | 布隆过滤器支持 |

## 五、代码实现详解

### 5.1 Redis 配置类

```java
package com.example.redis.config;

import org.springframework.ai.embedding.EmbeddingClient;
import org.springframework.ai.vectorstore.RedisVectorStore;
import org.springframework.ai.vectorstore.VectorStore;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.data.redis.core.RedisTemplate;

@Configuration
public class RedisConfig {

    @Value("${spring.ai.vectorstore.redis.index-name:my-index}")
    private String indexName;

    @Value("${spring.ai.vectorstore.redis.prefix:doc:}")
    private String prefix;

    @Bean
    public VectorStore vectorStore(EmbeddingClient embeddingClient, RedisTemplate<String, Object> redisTemplate) {
        return new RedisVectorStore(
            embeddingClient,
            redisTemplate,
            indexName,
            prefix
        );
    }
}
```

### 5.2 向量数据库服务类

```java
package com.example.redis.service;

import org.springframework.ai.document.Document;
import org.springframework.ai.vectorstore.SearchRequest;
import org.springframework.ai.vectorstore.VectorStore;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;

import java.util.List;
import java.util.Map;

@Service
public class RedisService {

    @Autowired
    private VectorStore vectorStore;

    public void addDocuments(List<Document> documents) {
        vectorStore.add(documents);
    }

    public List<Document> similaritySearch(String query, int topK) {
        return vectorStore.similaritySearch(query, topK);
    }

    public List<Document> similaritySearchWithFilter(String query, int topK, Map<String, Object> filter) {
        SearchRequest request = SearchRequest.query(query)
            .withTopK(topK)
            .withSimilarityThreshold(0.0)
            .withFilterExpression(filter);
        return vectorStore.similaritySearch(request);
    }

    public void deleteDocuments(List<String> ids) {
        vectorStore.delete(ids);
    }
}
```

### 5.3 REST API 控制器

```java
package com.example.redis.controller;

import org.springframework.ai.document.Document;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;
import com.example.redis.service.RedisService;

import java.util.ArrayList;
import java.util.List;
import java.util.Map;

@RestController
@RequestMapping("/api/redis")
public class RedisController {

    @Autowired
    private RedisService redisService;

    @PostMapping("/documents")
    public String addDocuments(@RequestBody List<Map<String, Object>> docs) {
        List<Document> documents = new ArrayList<>();
        for (Map<String, Object> doc : docs) {
            String content = (String) doc.get("content");
            @SuppressWarnings("unchecked")
            Map<String, Object> metadata = (Map<String, Object>) doc.get("metadata");
            documents.add(new Document(content, metadata));
        }
        redisService.addDocuments(documents);
        return "Documents added successfully";
    }

    @GetMapping("/search")
    public List<Document> search(
            @RequestParam String query,
            @RequestParam(defaultValue = "5") int topK) {
        return redisService.similaritySearch(query, topK);
    }

    @PostMapping("/search/filter")
    public List<Document> searchWithFilter(
            @RequestParam String query,
            @RequestParam(defaultValue = "5") int topK,
            @RequestBody Map<String, Object> filter) {
        return redisService.similaritySearchWithFilter(query, topK, filter);
    }

    @DeleteMapping("/documents")
    public String deleteDocuments(@RequestBody List<String> ids) {
        redisService.deleteDocuments(ids);
        return "Documents deleted successfully";
    }
}
```

## 十、常见问题

### Q1: Redis Vector Search 如何安装？

Redis Vector Search 是 Redis Stack 的一部分，推荐使用 Redis Stack Docker 镜像。如果自行编译，需要确保 RediSearch 模块已正确加载。

### Q2: 如何确保 Redis 的持久化？

Redis 支持 RDB（快照）和 AOF（追加文件）两种持久化方式。可以在 redis.conf 中配置：
- `save 60 1000`：60 秒内至少 1000 个键修改时触发快照
- `appendonly yes`：启用 AOF 持久化

### Q3: Redis 集群模式如何配置？

Spring Data Redis 支持 Redis 集群模式，需要在配置中指定集群节点：
```properties
spring.redis.cluster.nodes=host1:port1,host2:port2,host3:port3
```

## Java 客户端示例

以下是一个独立的 Java 客户端示例，用于调用 Spring AI Redis Vector Search 服务：

```java
import org.springframework.web.client.RestTemplate;
import org.springframework.http.*;
import java.util.Map;

public class RedisVectorClient {
    private final RestTemplate restTemplate;
    private final String baseUrl;

    public RedisVectorClient(String baseUrl) {
        this.baseUrl = baseUrl;
        this.restTemplate = new RestTemplate();
    }

    public Map<String, Object> search(String query) {
        String url = baseUrl + "/redis/search?query=" +
                    java.net.URLEncoder.encode(query, java.nio.charset.StandardCharsets.UTF_8);
        return restTemplate.getForObject(url, Map.class);
    }

    public Map<String, Object> addDocument(String text, Map<String, Object> metadata) {
        HttpHeaders headers = new HttpHeaders();
        headers.setContentType(MediaType.APPLICATION_JSON);

        Map<String, Object> body = Map.of(
            "text", text,
            "metadata", metadata
        );
        HttpEntity<Map<String, Object>> request = new HttpEntity<>(body, headers);

        ResponseEntity<Map> response = restTemplate.postForEntity(
            baseUrl + "/redis/add", request, Map.class);

        return response.getBody();
    }

    public static void main(String[] args) {
        RedisVectorClient client = new RedisVectorClient("http://localhost:8080");

        // 添加文档
        Map<String, Object> metadata = Map.of("category", "database", "type", "vector");
        Map<String, Object> addResult = client.addDocument(
            "Redis Vector Search 提供高性能向量搜索",
            metadata
        );
        System.out.println("Document added: " + addResult.get("id"));

        // 搜索文档
        Map<String, Object> searchResult = client.search("向量搜索");
        System.out.println("\nSearch results: " + searchResult.get("results"));
    }
}
```
